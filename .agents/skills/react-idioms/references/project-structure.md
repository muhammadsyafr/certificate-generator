## React Project Structure

React project structure follows the same vertical slice architecture as other frontend frameworks. See the relevant layout guide:

- **React SPA (Vite)**: See `@.agents/skills/vue-idioms/references/project-structure.md` — replace `.vue` with `.tsx`, composables with hooks, and Pinia stores with Zustand/Context
- **Next.js**: See `@.agents/skills/nextjs-idioms/references/project-structure.md`
- **Hono / Node.js Backend**: See `@.agents/skills/hono-idioms/references/project-structure.md`
- **General Frontend**: Follow feature-based vertical slices per `@.agents/rules/project-structure.md`
